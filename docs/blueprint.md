# **App Name**: OmniConvert Pro

## Core Features:

- Drag-and-Drop Universal Uploader: Seamless, zero-friction 'drag-and-drop' upload system for multiple video (MP4, MKV, AVI, MOV, WebM) and image (JPG, PNG, WebP, SVG, GIF, HEIC) formats via a large, interactive central drop zone that pulsates on hover.
- Dynamic Output Configuration & Tooltips: After upload, dynamic dropdown menus filter valid output formats based on the file type. Includes subtle info tooltips displaying technical details like codecs, resolution, and frame rate for various conversion options.
- Advanced Conversion Settings: Configure output settings such as target format (including bi-directional capabilities for all supported types), specific video codecs (H.264/H.265, ProRes), 4K/1080p resolution, frame rate, and bitrate for videos; and transparency toggles for images.
- Video-to-GIF Conversion: Dedicated functionality to directly convert uploaded video files into high-quality animated GIF clips.
- AI Smart Compression Tool: An intelligent tool for image conversions that utilizes AI models to determine optimal compression parameters for images, balancing file size reduction with visual fidelity based on selected quality levels.
- Real-time Conversion Progress: Provides detailed progress visualization with real-time percentages and status indicators for individual and batch conversion processes.
- Minimalist Download Center: Users can retrieve converted files individually, or as a single compressed ZIP archive if multiple files were converted (direct download for single converted files).

## Style Guidelines:

- Primary interactive color: An electric blue (#3D7BF5) with high saturation and vibrancy, creating a strong neon accent against dark backgrounds.
- Background color: A deep-space dark blue-grey (#181D25) derived from the primary hue, with heavy desaturation, to establish the core dark mode aesthetic.
- Accent color: A vibrant cyan (#75DBF0) to complement the electric blue, approximately 30 degrees counter-clockwise on the color wheel from the primary, offering clear contrast for highlights and secondary interactive elements.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly readable qualities, suitable for all UI text from labels to detailed information.
- Utilize minimalist line-art or filled icons with subtle gradients/glows that complement the neon and glassmorphism aesthetic, maintaining clarity and consistency.
- The layout features a minimalist header and places the conversion dashboard, centered around the large drop zone, as the primary focus. It employs glassmorphism card effects for content separation and maintains full responsiveness across desktop and mobile devices.
- Key animations include a pulsing effect on the central drag-and-drop zone upon hover, smooth transitions for dynamic dropdown menus and tooltips, and fluid progress animations for status indicators.
